# redirect-page-auto-closer-for-slack

Automatically closes Slack redirect pages and restores focus to the tab you were on beforehand. Supports Slack archive pages and app redirect pages. Easily change the countdown time by clicking one of the offered alternate times in blue.

## Safest extension. No Permissions Needed.

Use Slack for communication? Then this extension will keep your Chrome running fast and smooth by automatically closing Slack redirect pages after they've served their purpose and restoring focus to the tab you were on beforehand. 

The most user friendly Redirect Page Closer you'll find in the store that uses the least permissions while giving you the ability to cancel the auto close timer. We natively blend in with the Slack UI for a seamless experience. Auto closes Slack redirect tabs after detecting the redirect has occurred.

Complete privacy. Contains no analytics. Zero dependencies.

## Supported URLs
This extension automatically closes pages matching these patterns:
- `https://*.slack.com/archives/*` - Slack channel archive pages
- `https://*.slack.com/app_redirect*` - Slack app redirect pages
